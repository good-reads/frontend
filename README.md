## 기능
### 유저와 관련된 기능
- 회원가입, 로그인
- 프로필 이미지, 닉네임, 비밀번호 수정

### 책과 관련된 기능
**책 검색**
- 책 검색 시 lazy 하게 불러오도록 구현
- 책 검색 시 벽돌 레이아웃 적용

**책 등록**
- 책 등록 기능 구현

**리뷰 등록**
- 평점 및 100자평

### 서재(즐겨찾기)와 관련된 기능
- 서재 추가 및 삭제
- 책 서재에 추가 및 삭제
